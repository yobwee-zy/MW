# Luna Bloom Beauty Storefront

A modern, responsive ecommerce landing page for beauty products with:

- Hero and brand storytelling sections
- Category filters and product search
- Dynamic product cards
- Slide-in shopping cart with quantity tracking
- LocalStorage persistence for cart state

## Run Locally

1. Open the project in VS Code.
2. Use a static server (for example, Live Server extension) and open `index.html`.
3. Start shopping in the browser.

No build step or dependencies required.
